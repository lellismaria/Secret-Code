## 💻 Sobre o projeto

Esse é o resultado do primeiro Challenge da trilha Iniciante em Programação do programa Oracle Next Education.   

O desafio propõe substituir vogais por palavras e assim ser capaz de criptografar um texto. Além disso, também é possível descriptografar um texto criptografado. A letra "e" é convertida para "enter" A letra "i" é convertida para "imes" A letra "a" é convertida para "ai" A letra "o" é convertida para "ober" A letra "u" é convertida para "ufat".
